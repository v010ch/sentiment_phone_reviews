
 Сервис определения тональности русскоязычных отзывов на телефоны,
 представленный в виде docker образа, взаимодействующего с клиентской
 частью через python socket.
 Входные данные - текст отзыва.
 Выходные данные - тональность отзыва и подсветка слов оказавших наибольшее
                   влияние на установленную тональность.

 ## Воспроизведение инференса

   Выполнить команду
   sudo docker run -t --rm server_app:v(n)

   Перейти в дерикторию flask_app

   Выполнить команды
      set FLASK_APP=app
      flask run

   Перейти в браузере по алресу http://127.0.0.1:5000/

 ## Техническая реализация инференса
 
Инференс реализован по средствам onnx, обернутого в docker, 
взаимодействующего с клиентской частью через python socket.

Клиентская часть реализована на flask, взаимодействующей с
серверной (docker) частью через python socket.

На выбор представлены 4 варианта моделей от более быстрых
для инференса и менее требовательных к ресурсам logreg и 
catboost, но менее точных, до более тяжелых и медленных, но 
с лучшим качеством определения тональности трансформеров с 
hugging face.

Для легковестных моделей дополнительно существует возможность
подсвечивать слова отзыва, наиболее повлиявшие на полученную 
тональность.

Отзывы на русском с возможным вкроплением отдельных английских 
терминов. Максимальная обрабатываемая длина - 96 слов без учета 
спецсимволов/цифр. Длинные отзывы позволяют лучше определить 
тональности.

 ## Данные

Сбор отзывов оcуществлялся с [портала](https://irecommend.ru) по 
средствам [selenium](https://www.selenium.dev/).

Класс тональности (таргет) определялся по оценка телефона, 
выставленной на портале: 1, 2 и 3 звезды считал негативным отзывом;
4 и 5 - позитивным.

В качестве отложенной выборки были выбраны данные с соревнования
[kaggle inclass](https://www.kaggle.com/c/simplesentiment/overview). В которых предоставлены отзывы на телефоны, но в
количестве недостаточном для обучения моделей. Таргет - 2 тональности:негативная и позитивная.
